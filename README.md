# BookMYRoom


## Problem Statement
Institute often face challenges in managing classroom, laboratory, and auditorium bookings. Manual booking processes or uncoordinated scheduling can lead to conflicts, double bookings, and inefficient utilization of rooms.

The goal of this project is to develop a **web-based room booking system** that allows students and faculty to request rooms while administrators manage approvals and room availability. The system prevents scheduling conflicts, supports hierarchical approvals, and provides real-time booking status updates.

The system follows a **Role-Based Access Control (RBAC)** model where different users such as **students, faculty, room administrators, and administrators** have specific permissions and responsibilities.
